# AlphaFlow Status Check

## Quick Commands

```bash
du -sh .                    # Project size
git status                  # Git state
git log --oneline -5        # Recent commits
openspec list               # Active changes
npm run build               # Build check
```

## Health Verification

- Build: npm run build
- Types: npm run typecheck
- Lint: npm run lint
- Server: npm run dev (check Alpaca connects)
