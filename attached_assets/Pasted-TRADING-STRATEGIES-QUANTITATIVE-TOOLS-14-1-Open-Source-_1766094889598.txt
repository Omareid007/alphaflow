14.3 Strategy Metrics & KPIs
Core Performance Metrics:

interface StrategyMetrics {
  // Returns
  totalReturn: number;           // Total percentage return
  annualizedReturn: number;      // CAGR

  // Risk-Adjusted
  sharpeRatio: number;           // (Return - RiskFree) / StdDev, target > 1.0
  sortinoRatio: number;          // Only penalizes downside volatility
  calmarRatio: number;           // Return / MaxDrawdown, target > 0.5

  // Risk
  maxDrawdown: number;           // Largest peak-to-trough decline
  volatility: number;            // Standard deviation of returns

  // Trading
  winRate: number;               // Winning trades / Total trades
  profitFactor: number;          // Gross profit / Gross loss, target > 1.5
  expectancy: number;            // (WinRate × AvgWin) - (LossRate × AvgLoss)

  // Efficiency
  tradesPerMonth: number;
  avgHoldingPeriod: number;      // Days
}
Kelly Criterion for Position Sizing:

// Optimal position size formula
function kellyPosition(winRate: number, winLossRatio: number): number {
  // f* = (bp - q) / b
  // b = win/loss ratio, p = win probability, q = lose probability
  const kelly = (winLossRatio * winRate - (1 - winRate)) / winLossRatio;

  // Use fractional Kelly (25-50%) for safety
  return Math.max(0, Math.min(kelly * 0.25, 0.10)); // Cap at 10%
}
14.4 Research Papers & Proven Strategies
Momentum Strategies:

Optimal window: 9-12 months moving average
Works better on volatile stocks (Tesla, crypto)
Combine with trend confirmation
Mean Reversion:

TSI-based mean reversion (arXiv 2024)
Works better on stable assets
2 standard deviations from mean as entry trigger
Fama-French Factor Models:

Data: https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/Data_Library/
Five factors: Market, Size, Value, Profitability, Investment
Explains 90%+ of portfolio returns
Machine Learning Resources:

Coverage: Linear regression to deep RL, NLP for 10-Ks
